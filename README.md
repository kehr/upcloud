# upcloud

这个项目的目标是实现`云空间本地化管理`。你可以用本项目提供的操作同步或备份你的云空间数据。

##安装  

```bash
pip install upcloud
```
##使用 

```bash
upcloud -b BUCKET -u USERNAME -p PASSWD
```
获取命令帮助:`command [-h|--help]`  
获取详细说明:`man [command]`

##特色  

1. 支持命令历史回滚

2. 支持<Tab>键命令补全  

3. 支持<Tab> <Tab> 显示命令列表

4. 支持执行shell命令  

5. 支持清屏操作  
......  

以上特性已经完成，其它小特性待完成后补充。总之，就是让你感觉操作云空间就像操作你本地空间一样爽快。这也是这个项目的目标和动力。

正在实现，当前目录下文件名自动补全。这个我觉得比较给力～
