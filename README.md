支持操作：

1. ls  列出空间所有文件，默认选项是 `-a`  
   ls -a               列出当前工作目录下的所有文件        
   ls -f [file name]   查看指定文件/文件夹的属性  
   ls -s               查看当前空间的信息  

2. cd 修改当前操作目录  
   cd [dir] 修改当前操作的目录为dir 

3. pwd 查看当前工作目录  

4. mv 移动空间中的文件或目录  
   mv [src] [des]  移动指定文件到目的地。支持移动和重命名操作。

5. up 上传指定文件  
   up [src] [des] 上传本地文件到云空间指定位置，如果目标路径没有指定，则上传到根目录下

6. down 下载指定文件  
   down [file path] 下载指定路径下的文件 

7. rm 删除指定文件  
   rm [file path]  删除指定路径下的文件
